## 图标规格
- **尺寸**：81px × 81px（微信小程序标准尺寸）
- **格式**：PNG
- **大小**：均小于 40kb

## 更新图标

如需更新图标，请确保新图标符合以下要求：
- 尺寸：81px × 81px
- 格式：PNG（支持透明背景）
- 大小：不超过 40kb

可使用以下命令批量调整图标尺寸：
```bash
sips -z 81 81 图标文件.png -s format png
```
